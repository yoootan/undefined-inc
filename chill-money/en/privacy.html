Privacy Policy

Last updated: May 20, 2025

UNDEFINED, K.K. ("we", "our", or "the Company") sets forth this Privacy Policy regarding the handling of information in "Chill Money" (the "App").

1. Information We Handle

The App handles the following information:

  • Asset information entered by the user (asset names, categories, amounts, etc.)
  • Asset history records (snapshots)

The App does not require account registration. No personal information such as email addresses or names is collected.

2. Where Data Is Stored

Data entered by the user is stored only in the following locations:

  • The user's device (local storage)
  • iCloud (when the user has enabled iCloud sync)

No data is transmitted to or stored on servers operated by the Company.

3. Third-Party Sharing

Since the App does not transmit user data to external servers, no data is shared with third parties.

4. External Services

The App uses the following external services:

  • Apple iCloud (data synchronization)
  • Exchange rate API (anonymous access for currency conversion)

No personally identifiable information is transmitted to these services.

5. Analytics & Tracking

The App does not use any analytics tools or tracking SDKs to monitor user behavior.

6. Data Security

On-device data is protected by the operating system's security features. iCloud data is protected by Apple's encryption technology. The App also offers Face ID app lock functionality.

7. Data Deletion

Users can delete individual asset data from within the App. Uninstalling the App will remove all data from the device. iCloud data can be removed by managing iCloud storage in device settings.

8. Changes to This Policy

We may update this Privacy Policy from time to time. Significant changes will be communicated through the App.
